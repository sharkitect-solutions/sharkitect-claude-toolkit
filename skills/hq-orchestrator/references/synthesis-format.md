# Synthesis Output Format

When combining results from 2+ agents, use this structure. The goal is a SINGLE executive-ready output that enables the user to make a decision without reading each agent's raw output.

## Template

```markdown
## [Task Title] — Orchestrated Assessment

### Recommendation
[1-3 sentences. Lead with the action. Be decisive.]

### Agent Consensus
[What ALL agents agreed on. Bullet list.]

### Divergent Views
[Where agents disagreed. Present BOTH positions with reasoning.]
- **[Agent A] says:** [position + reasoning]
- **[Agent B] says:** [position + reasoning]
- **Orchestrator assessment:** [which position is stronger and why]

### Key Numbers
| Metric | Value | Source Agent |
|--------|-------|-------------|
| [e.g., Revenue impact] | [e.g., +$12K/yr] | financial-analyst |
| [e.g., Risk score] | [e.g., 7/16 MODERATE] | legal-advisor |
| [e.g., Timeline] | [e.g., 3 weeks] | project-manager |

### Decision Options
1. **[Recommended option]** — [1 sentence why]
2. **[Alternative option]** — [1 sentence tradeoff]
3. **[Conservative option]** — [1 sentence when this is better]

### Flagged Items
- [ ] [Any items needing user action or external input]
- [ ] [Any follow-up tasks generated by this assessment]
```

## Synthesis Rules

1. **Never copy-paste raw agent output.** Synthesize into executive language.
2. **Numbers beat adjectives.** Say "$4,200 margin" not "good margin."
3. **Disagreements are valuable.** Present them clearly — they reveal tradeoffs the user needs to weigh.
4. **Rank options.** The orchestrator has context across all agents — use it to rank, not just list.
5. **Flag gaps.** If any agent couldn't complete its assessment due to missing data, say so explicitly.
6. **Time-bound recommendations.** If the recommendation changes based on timing (e.g., "act within 7 days"), say so.

## Synthesis Depth Tiers

| Tier | When | Format |
|------|------|--------|
| **Quick** | 2 agents, low stakes, clear consensus | 3-5 bullet summary, no table |
| **Standard** | 2-4 agents, moderate stakes, some divergence | Full template above |
| **Executive** | 4+ agents, high stakes, significant divergence | Full template + risk matrix + scenario analysis |
