# Cold Email Copywriting Frameworks

## PAS (Problem, Agitate, Solution)

Identify pain -> Amplify consequences -> Present solution + soft CTA.

**Best for:** Problem-aware but not solution-aware prospects. The workhorse framework.

**Structure:**
1. Name the problem they're experiencing
2. Agitate: show what happens if it persists
3. Present your solution briefly + CTA

## BAB (Before, After, Bridge)

Current painful situation -> Ideal future -> Your product as the bridge.

**Best for:** Transformation-driven offers with emotional decision-makers.

## QVC (Question, Value, CTA)

Targeted pain question -> Brief value -> Direct next step.

**Best for:** C-suite prospects preferring brevity.

## AIDA (Attention, Interest, Desire, Action)

Hook/stat -> Address specific challenge -> Social proof/outcome -> Clear CTA.

**Best for:** Data-driven prospects and high-ticket pitches.

## PPP (Praise, Picture, Push)

Genuine compliment -> How things could be better -> Gentle push to action.

**Best for:** Senior prospects who respond to relationship-building.

## Star-Story-Solution

Introduce character -> Tell challenge narrative -> Reveal results.

**Best for:** Humanizing pitches through customer success stories.

## SCQ (Situation, Complication, Question)

Current reality -> Complicating challenge -> Question that speaks to need.

**Best for:** Mirrors consultative selling approaches.

## ACCA (Awareness, Comprehension, Conviction, Action)

Contrarian hook -> Explain benefit simply -> Provide proof -> Strong CTA.

**Best for:** Analytical buyers.

## 3C's (Alex Berman)

Compliment -> Case Study -> CTA.

**Best for:** Agency and services outreach.

## Mouse Trap (Will Allred)

Observation + Binary value-prop question in 1-2 sentences.

**Best for:** Maximum brevity and curiosity-driven replies.

## Justin Michael Method

Trigger/Pain -> Solution hint -> Binary CTA. 1-3 sentences, no intro.

**Best for:** High-velocity SDR teams at scale.

## Vanilla Ice Cream (Will Allred)

Observation -> Problem/Insight -> Credibility -> Solution -> Call-to-Conversation.

**Best for:** Universal base framework.

## PASTOR

Problem -> Amplify -> Story -> Testimony -> Offer -> Response.

**Best for:** Longer sequences and complex B2B services.
