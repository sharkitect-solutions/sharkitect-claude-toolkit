# Agent Instructions

You operate inside the **WAT framework** (Workflows, Agents, Tools). Probabilistic AI handles reasoning; deterministic code handles execution. That separation is what makes this system reliable.

**Layers:**
- **Workflows** (`workflows/`): Markdown SOPs — objectives, inputs, tools to use, outputs, edge cases
- **Agent** (you): Read workflows, run tools in sequence, handle failures, ask when unclear
- **Tools** (`tools/`): Python scripts for API calls, data transforms, file ops. Credentials in `.env`

**Core principle:** Each AI reasoning step is ~90% accurate. Five steps = 59%. Offload execution to deterministic scripts to stay reliable.

## Section 7: Bootstrap Tool

> **INSTANTIATION BLUEPRINT** — Copy this source code verbatim to `tools/bootstrap.py` during instantiation.

```python
"""
WAT Framework Bootstrap Tool
Checks environment, detects Superpowers, inventories installed skills/agents/plugins.
Pure Python stdlib — no external dependencies required.
"""

import json
import platform
import re
import subprocess
from datetime import datetime, timezone
from pathlib import Path
from typing import Any


def get_claude_home():
    """Return the ~/.claude/ directory path."""
    return Path.home() / ".claude"


def check_command(cmd):
    """Check if a command exists on PATH. Returns version string or None."""
    try:
        result = subprocess.run(
            cmd, capture_output=True, text=True, timeout=10,
            shell=(platform.system() == "Windows")
        )
        output = result.stdout.strip() or result.stderr.strip()
        return output if result.returncode == 0 else None
    except (FileNotFoundError, subprocess.TimeoutExpired, OSError):
        return None


def detect_platform() -> dict[str, Any]:
    """Gather platform information."""
    node_ver = check_command(["node", "--version"])
    npm_ver = check_command(["npm", "--version"])
    npx_ver = check_command(["npx", "--version"])

    return {
        "os": platform.system(),
        "os_version": platform.version(),
        "architecture": platform.machine(),
        "python_version": platform.python_version(),
        "home_directory": str(Path.home()),
        "claude_home": str(get_claude_home()),
        "node_available": node_ver is not None,
        "node_version": node_ver,
        "npm_available": npm_ver is not None,
        "npm_version": npm_ver,
        "npx_available": npx_ver is not None,
        "npx_version": npx_ver,
        "marketplace_ready": npx_ver is not None,
    }


def check_superpowers(claude_home):
    """Check if Superpowers plugin is installed. Returns status dict."""
    result = {
        "installed": False,
        "version": None,
        "install_path": None,
        "marketplace_registered": False,
        "install_commands": [],
    }

    plugins_file = claude_home / "plugins" / "installed_plugins.json"
    if not plugins_file.exists():
        result["install_commands"] = [
            "/plugin marketplace add anthropics/claude-plugins-official",
            "/plugin install superpowers@claude-plugins-official",
        ]
        return result

    try:
        data = json.loads(plugins_file.read_text(encoding="utf-8"))
        plugins = data.get("plugins", {})

        if "superpowers@claude-plugins-official" in plugins:
            entries = plugins["superpowers@claude-plugins-official"]
            if entries:
                entry = entries[0]
                result["installed"] = True
                result["version"] = entry.get("version")
                result["install_path"] = entry.get("installPath")
        else:
            result["install_commands"].append(
                "/plugin install superpowers@claude-plugins-official"
            )
    except (json.JSONDecodeError, KeyError):
        result["install_commands"].append(
            "/plugin install superpowers@claude-plugins-official"
        )

    marketplaces_file = claude_home / "plugins" / "known_marketplaces.json"
    if marketplaces_file.exists():
        try:
            mp_data = json.loads(marketplaces_file.read_text(encoding="utf-8"))
            if "claude-plugins-official" in mp_data:
                result["marketplace_registered"] = True
            else:
                result["install_commands"].insert(
                    0, "/plugin marketplace add anthropics/claude-plugins-official"
                )
        except (json.JSONDecodeError, KeyError):
            pass
    else:
        result["install_commands"].insert(
            0, "/plugin marketplace add anthropics/claude-plugins-official"
        )

    return result


def inventory_plugins(claude_home):
    """Read all installed plugins from installed_plugins.json."""
    plugins_file = claude_home / "plugins" / "installed_plugins.json"
    inventory = []

    if not plugins_file.exists():
        return inventory

    try:
        data = json.loads(plugins_file.read_text(encoding="utf-8"))
        plugins = data.get("plugins", {})

        for key, entries in plugins.items():
            if not entries:
                continue
            entry = entries[0]
            name, _, marketplace = key.partition("@")
            inventory.append({
                "name": name,
                "marketplace": marketplace,
                "key": key,
                "version": entry.get("version"),
                "scope": entry.get("scope"),
                "install_path": entry.get("installPath"),
            })
    except (json.JSONDecodeError, KeyError):
        pass

    return inventory


def parse_yaml_frontmatter(filepath):
    """Extract name and description from YAML frontmatter in a markdown file."""
    try:
        content = filepath.read_text(encoding="utf-8")
    except (OSError, UnicodeDecodeError):
        return None

    match = re.match(r"^---\s*\n(.*?)\n---", content, re.DOTALL)
    if not match:
        return None

    frontmatter = match.group(1)
    result = {}

    for line in frontmatter.split("\n"):
        line = line.strip()
        if line.startswith("name:"):
            result["name"] = line[5:].strip().strip("\"'")
        elif line.startswith("description:"):
            desc = line[12:].strip()
            if (desc.startswith('"') and desc.endswith('"')) or \
               (desc.startswith("'") and desc.endswith("'")):
                desc = desc[1:-1]
            result["description"] = desc

    return result if "name" in result else None


def inventory_skills(claude_home):
    """Scan ~/.claude/skills/ for all SKILL.md files and parse frontmatter."""
    skills_dir = claude_home / "skills"
    inventory = []

    if not skills_dir.exists():
        return inventory

    for skill_file in skills_dir.glob("*/SKILL.md"):
        parsed = parse_yaml_frontmatter(skill_file)
        if parsed:
            parsed["path"] = str(skill_file)
            parsed["source"] = "local"
            inventory.append(parsed)

    return inventory


def inventory_agents(claude_home):
    """Scan ~/.claude/agents/ for all .md files and parse frontmatter."""
    agents_dir = claude_home / "agents"
    inventory = []

    if not agents_dir.exists():
        return inventory

    for agent_file in agents_dir.glob("*.md"):
        parsed = parse_yaml_frontmatter(agent_file)
        if parsed:
            parsed["path"] = str(agent_file)
            parsed["source"] = "local"
            inventory.append(parsed)

    return inventory


def inventory_marketplaces(claude_home):
    """Read registered marketplaces."""
    mp_file = claude_home / "plugins" / "known_marketplaces.json"
    marketplaces = []

    if not mp_file.exists():
        return marketplaces

    try:
        data = json.loads(mp_file.read_text(encoding="utf-8"))
        for name, info in data.items():
            marketplaces.append({
                "name": name,
                "repo": info.get("source", {}).get("repo", "unknown"),
                "install_location": info.get("installLocation"),
            })
    except (json.JSONDecodeError, KeyError):
        pass

    return marketplaces


def inventory_rules(claude_home):
    """Scan ~/.claude/rules/ for all .md files."""
    rules_dir = claude_home / "rules"
    inventory = []
    if not rules_dir.exists():
        return inventory
    for rule_file in rules_dir.glob("*.md"):
        inventory.append({
            "name": rule_file.stem,
            "path": str(rule_file),
        })
    return inventory


def write_json(filepath, data):
    """Write JSON data to file, creating parent directories if needed."""
    filepath.parent.mkdir(parents=True, exist_ok=True)
    filepath.write_text(json.dumps(data, indent=2, default=str), encoding="utf-8")


def main():
    project_root = Path.cwd()
    tmp_dir = project_root / ".tmp"
    tmp_dir.mkdir(exist_ok=True)

    sentinel = tmp_dir / ".bootstrap-complete"
    if sentinel.exists():
        print("Bootstrap already complete. Delete .tmp/.bootstrap-complete to re-run.")
        return

    claude_home = get_claude_home()
    timestamp = datetime.now(timezone.utc).isoformat()

    print("=" * 60)
    print("WAT Framework Bootstrap")
    print("=" * 60)

    # Phase 1: Platform Detection
    print("\n[Phase 1] Detecting platform...")
    platform_info = detect_platform()
    write_json(tmp_dir / "platform.json", platform_info)

    print(f"  OS: {platform_info['os']} {platform_info['os_version']}")
    print(f"  Python: {platform_info['python_version']}")
    print(f"  Node.js: {platform_info['node_version'] or 'NOT FOUND'}")
    print(f"  npx: {platform_info['npx_version'] or 'NOT FOUND'}")
    print(f"  Marketplace ready: {platform_info['marketplace_ready']}")

    # Phase 2: Superpowers Check
    print("\n[Phase 2] Checking Superpowers...")
    superpowers = check_superpowers(claude_home)

    if superpowers["installed"]:
        print(f"  Superpowers: INSTALLED (v{superpowers['version']})")
    else:
        print("  Superpowers: NOT INSTALLED")
        print("  Required commands:")
        for cmd in superpowers["install_commands"]:
            print(f"    {cmd}")

    # Phase 3: Dynamic Inventory
    print("\n[Phase 3] Inventorying installed components...")

    skills = inventory_skills(claude_home)
    agents = inventory_agents(claude_home)
    plugins = inventory_plugins(claude_home)
    marketplaces = inventory_marketplaces(claude_home)
    rules = inventory_rules(claude_home)

    print(f"  Skills found: {len(skills)}")
    print(f"  Agents found: {len(agents)}")
    print(f"  Plugins found: {len(plugins)}")
    print(f"  Marketplaces registered: {len(marketplaces)}")
    print(f"  Rules found: {len(rules)}")

    inventory = {
        "generated_at": timestamp,
        "skills": skills,
        "agents": agents,
        "plugins": plugins,
        "marketplaces": marketplaces,
        "rules": rules,
    }
    write_json(tmp_dir / "available-inventory.json", inventory)

    # Phase 4: Generate Report
    print("\n[Phase 4] Generating bootstrap report...")

    report = {
        "generated_at": timestamp,
        "platform": platform_info,
        "superpowers": superpowers,
        "inventory_counts": {
            "skills": len(skills),
            "agents": len(agents),
            "plugins": len(plugins),
            "marketplaces": len(marketplaces),
            "rules": len(rules),
        },
        "actions_needed": [],
    }

    if not superpowers["installed"]:
        report["actions_needed"].append({
            "priority": "CRITICAL",
            "action": "Install Superpowers",
            "commands": superpowers["install_commands"],
            "reason": "Superpowers enables auto-triggered brainstorming, TDD, debugging, and code review workflows.",
        })

    if not platform_info["npx_available"]:
        report["actions_needed"].append({
            "priority": "RECOMMENDED",
            "action": "Install Node.js",
            "reason": "Required for marketplace search during skills evaluation. Install from https://nodejs.org/",
        })

    # Check for universal protocols rule
    universal_rule = claude_home / "rules" / "universal-protocols.md"
    if not universal_rule.exists():
        report["actions_needed"].append({
            "priority": "CRITICAL",
            "action": "Install Universal Protocols",
            "reason": "Universal protocols rule not found at ~/.claude/rules/universal-protocols.md. "
                      "Install from toolkit repo: copy rules/universal-protocols.md to ~/.claude/rules/",
        })

    write_json(tmp_dir / "bootstrap-report.json", report)

    sentinel.write_text(json.dumps({"completed_at": timestamp}), encoding="utf-8")

    # Summary
    print("\n" + "=" * 60)
    print("Bootstrap Complete")
    print("=" * 60)

    if report["actions_needed"]:
        print("\nActions needed:")
        for action in report["actions_needed"]:
            print(f"\n  [{action['priority']}] {action['action']}")
            print(f"  Reason: {action['reason']}")
            if "commands" in action:
                print("  Commands:")
                for cmd in action["commands"]:
                    print(f"    {cmd}")
    else:
        print("\nAll systems ready. No actions needed.")

    print(f"\nReports written to: {tmp_dir}")
    print("  - platform.json")
    print("  - available-inventory.json")
    print("  - bootstrap-report.json")


if __name__ == "__main__":
    main()
```

```markdown
# Skills Evaluation Workflow

## Objective
Given a project purpose, dynamically discover what capabilities exist on this machine, match them to the project needs, search external marketplaces for gaps, and install approved components globally.

## Trigger
`PROJECT_PURPOSE` is set in CLAUDE.md and `.tmp/skills-manifest.json` does not exist.

## Inputs
- `PROJECT_PURPOSE` from CLAUDE.md
- `.tmp/available-inventory.json` (generated by bootstrap)
- `.tmp/platform.json` (generated by bootstrap)

## Procedure

### Step 1: Parse Project Purpose
Break the project purpose into categories:
- **Primary domain** (e.g., lead-generation, e-commerce, data-pipeline, content-marketing)
- **Key activities** (e.g., email-writing, web-scraping, API-building, research, automation)
- **Output types** (e.g., reports, emails, dashboards, automations, documents)
- **Technologies likely needed** (e.g., APIs, databases, web scraping, AI/LLM)

### Step 2: Score Installed Skills
Read `.tmp/available-inventory.json` and examine the `skills` array.
For each installed skill:
- Compare its `name` and `description` against the project categories from Step 1
- Score relevance on a 0-10 scale:
  - **7-10**: Directly relevant — recommend for this project
  - **4-6**: Possibly useful — mention as optional
  - **0-3**: Not relevant — skip

### Step 3: Score Installed Agents
Same scoring process for each entry in the `agents` array.

### Step 4: Score Installed Plugins
For each entry in the `plugins` array (beyond superpowers):
- Assess relevance to the project purpose
- Note which are currently active

### Step 5: Identify Gaps
Based on the project purpose, identify capabilities NOT covered by any installed component:
- Missing skill categories
- Missing agent types
- Missing plugin functionality
- Missing MCP server connections

### Step 5b: Validate Universal Protocols
Check if `~/.claude/rules/universal-protocols.md` exists:
- If YES: PASS (universal protocols will auto-load in every session)
- If NO: Flag as critical gap. Install from toolkit repo:
  `cp sharkitect-claude-toolkit/rules/universal-protocols.md ~/.claude/rules/`
  Or warn user to run INSTALL-GUIDE.md Step 7.

### Step 6: Search External Marketplace
Check `.tmp/platform.json` — if `marketplace_ready` is `true`:

1. Use WebFetch to browse marketplace categories matching the project domain:
   - https://www.aitmpl.com/skills
   - https://www.aitmpl.com/agents
   - https://www.aitmpl.com/hooks
   - https://www.aitmpl.com/mcps
2. Cross-reference marketplace offerings against the gaps identified in Step 5
3. Note install commands for relevant components:
   - Skills: `npx claude-code-templates@latest --skills <name> --yes`
   - Agents: `npx claude-code-templates@latest --agents <name> --yes`

If `marketplace_ready` is `false`:
- Skip marketplace search
- Note: "Install Node.js to enable marketplace component discovery"

If aitmpl.com is unreachable:
- Skip external search
- Note it in the report, continue with installed inventory only

### Step 7: Check Plugin Marketplaces
Also evaluate plugins from known Claude Code plugin marketplaces:
- `claude-plugins-official` (anthropics/claude-plugins-official)
- `claude-code-workflows` (wshobson/agents)
- `agentic-forge` (e-stpierre/agentic-forge)

Identify useful plugins not yet installed that would benefit this project.

### Step 8: Present Recommendations
Format recommendations in three sections:

**Already Available (Use These)**
| Component | Type | Score | Why It Fits This Project |

**Recommended New Installs**
| Component | Type | Source | Install Command | Why It's Needed |

**Optional / Nice-to-Have**
| Component | Type | Source | Install Command | Why It Could Help |

### Step 9: Get User Approval
Present the full recommendation tables to the user.
Wait for the user to approve or reject each recommendation.
**Never install anything without explicit user approval.**

### Step 10: Execute Approved Installs
For approved marketplace components:
- `npx claude-code-templates@latest --skills <name> --yes`
- `npx claude-code-templates@latest --agents <name> --yes`

For approved plugins:
- `/plugin marketplace add <org/repo>` (if marketplace not yet registered)
- `/plugin install <name>@<marketplace>`

**All installs are GLOBAL** — they go to `~/.claude/` and are available to every project on this machine.
```
